# 法务周报看板

这是一个本地静态前端原型，用于验证“法务周报看板”的展示架构。当前视觉版本参考了数据合规看板的深色工作台风格。

## 打开方式

直接双击 `index.html`，或在浏览器中打开：

`/Users/chenxue/Documents/法务周报看板/index.html`

## 当前范围

- 总览页展示三块内容：本周亮点成果概览、重要风险提示、运营数据。
- 页面采用顶部视觉区、固定模块导航、浅色工作台数据面板和响应式布局。
- 右上角周次支持下拉筛选，切换后会跳转到对应 `?week=` 周报视图。
- 八个导航模块包括：总览、项目支持、数据合规、战投与公司治理、IP资产与保护、争议与人事风险、运营与组织能力、职能支持。
- 详情页展示本周事项、状态筛选和历史时间轴。
